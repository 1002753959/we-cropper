## 演示
![](https://github.com/dlhandsome/we-cropper/blob/master/example.gif?raw=true)
## data参数说明


| 属性 | 类型 | 默认值 | 说明 |
| ---- | ---- | ---- | ---- |
| id | String | 'cropper' | canvas组件标识符 |
| width | String | - | 裁剪图片区域宽度 |
| height | String | - | 裁剪图片区域高度 |
| minScale | String | - | 最小缩放倍数 |
| maxScale | String | - | 最大缩放倍数 |
| src | String | - | 需裁剪的图片资源 |
